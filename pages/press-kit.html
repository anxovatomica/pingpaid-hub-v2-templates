One-Paragraph Description

PingPaid is a suite of free, browser-based tools for freelancers and small businesses. It includes an invoice generator (with PDF export and logo upload), a late fee calculator, an hourly rate calculator, a break-even calculator, an interest calculator, a tip calculator, and a percentage calculator. All tools work instantly with no signup, no watermarks, and no ads. Everything is open source and hosted on GitHub Pages.

Two-Sentence Description

PingPaid offers 7 free tools that help freelancers invoice clients, calculate rates, and manage finances. Built with vanilla HTML/CSS/JS, the tools require no signup and work entirely in the browser.
